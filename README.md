# Butterfly Effect Legacy Supabase Backend

> This branch preserves the earlier Supabase Edge Functions backend for Butterfly Effect.
> The current production app lives on `main` and uses Node.js/Express on AWS EC2 with React 18 streaming SSR.

### What This Branch Contains

- `frontend/`: earlier React client implementation
- `supabase/functions/`: Supabase Edge Function backend write path
